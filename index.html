Context

It is very common for webpages to have several variations of their page in order to see what modifications or updates users prefer most. For example, have you and your friends had different updates of Instagram? I have. My feed still shows the number of likes in each picture, while that of my roommate does not. These two versions of the same page are there to test user preference and their actions.

For this project, we were given a control metric and asked to modify it to create a second version of it. The control was option A, and the variation was B. After the modifications were done, we collected data and analyzed it.

Usability & Accessibility Issues

  • Website is not responsive, meaning that its appearance does not chance according to screen size or device.

  • There is no information about the business.

  • There are no recent menus, apparently they updated them but not the website.

  • They have several phone numbers on the page that are not valid anymore.

  • There is a slogan that looks like a navigation bar which is confusing for the user.

  • Images are too small to properly be appreciated and viewed.
